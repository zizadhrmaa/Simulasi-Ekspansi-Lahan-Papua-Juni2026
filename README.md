# Simulasi-Ekspansi-Lahan-Papua-Juni2026
Simulasi ekspansi lahan di Papua dengan algoritma Greedy. Dibuat dalam rangka pengerjaan makalah berjudul "Simulasi Ekspansi Lahan Perkebunan Menggunakan Algoritma Greedy dan Analisis Dampaknya terhadap Fragmentasi Hutan Adat Papua" untuk mata kuliah Strategi Algoritma Institut Teknologi Bandung Tahun Ajaran 2025/2026.
